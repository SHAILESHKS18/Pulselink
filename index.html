Project Overview

Our project is designed to develop a real-time ECG (Electrocardiogram) monitoring system using a Raspberry Pi. The system captures ECG readings from an ad8232 via mcp3008 and processes the data for detecting any abnormalities and at the same time acting as a backend server to host an website(github repositary).


Processed ECG readings are displayed on a dedicated website, providing an intuitive, user-friendly platform for real-time heart monitoring.


Future enhancements include data logging, tracking historical ECG readings, and mobile access, improving accessibility and security in medical data management.
